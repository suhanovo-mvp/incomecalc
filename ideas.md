# Идеи дизайна для Balance Counter

## Выбранный подход: Modern Financial Dashboard

### Design Movement
**Современный финтех-минимализм** с элементами кинетического дизайна. Вдохновлено интерфейсами премиум-приложений для управления финансами (Revolut, N26).

### Core Principles
1. **Кинетическая визуализация** - каждое изменение баланса должно быть видимо и ощущаемо через плавные анимации
2. **Иерархия через контраст** - основной счётчик доминирует, вспомогательная информация подчинена
3. **Органическая глубина** - использование мягких теней и слоистости вместо плоских элементов
4. **Реактивность** - интерфейс реагирует на каждое обновление баланса визуально

### Color Philosophy
- **Основной цвет**: Глубокий изумрудный (Emerald-600: `#059669`) - символизирует рост и процветание финансов
- **Акцент**: Золотой (Amber-400: `#fbbf24`) - привлекает внимание к основному счётчику
- **Фон**: Очень светлый серо-голубой (`#f8fafc`) - спокойствие и профессионализм
- **Текст**: Темный сланец (`#1e293b`) - высокий контраст для читаемости
- **Вторичные элементы**: Серый (`#64748b`) - поддерживающая информация

### Layout Paradigm
- **Асимметричная композиция**: Основной счётчик смещён в верхнюю часть с большим пространством вокруг
- **Плавающие карточки**: Информационные элементы расположены как слои на разных уровнях глубины
- **Динамическое пространство**: Счётчик занимает центральную зону с радиальной симметрией

### Signature Elements
1. **Пульсирующий счётчик** - основной элемент с мягким свечением и анимацией
2. **Орбитальные индикаторы** - маленькие элементы, вращающиеся вокруг основного счётчика
3. **Линейный прогресс** - тонкая анимированная линия, показывающая прогресс в течение месяца

### Interaction Philosophy
- Каждое обновление баланса сопровождается визуальной реакцией (масштабирование, свечение)
- Наведение на элементы раскрывает дополнительную информацию
- Плавные переходы между состояниями

### Animation Guidelines
- **Обновление баланса**: 0.6s ease-out масштабирование + изменение цвета
- **Пульсирование**: Бесконечная анимация с периодом 2s, амплитуда 5%
- **Вращение орбиты**: 8s linear бесконечное вращение
- **Появление элементов**: 0.8s ease-out fade-in с небольшим сдвигом вверх

### Typography System
- **Заголовок**: `Geist` (Google Fonts) - 700 weight, 3.5rem - мощный и современный
- **Основной текст**: `Inter` - 400 weight, 1rem - читаемость
- **Числовые данные**: `IBM Plex Mono` - 600 weight - точность и финтех-стиль
- **Вспомогательный текст**: `Inter` - 400 weight, 0.875rem, серый цвет
